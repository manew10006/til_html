header 영역이 있다.

이 영역의 '''내용별''' 레이아웃을 먼저 고민해 보자.
이때 사용하는 html 태그는 div 입니다.

내용 + 디자인 + 기능 = 컴포넌트

class="name*inname" 태그의 이름을 짓는데 - 는 연산자로 쓰이고, 그냥 공백을 하면 이름을 2개 부여하기 때문에 * 언더바로 하나로 짓는다. (언더바 단계는 4단계까지 가지않는다. 3단계 까지만.)
